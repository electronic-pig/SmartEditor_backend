# ✨ 简介

**妙笔** —— 基于大小模型的在线文档富文本编辑器

> 2024年中国软件杯A10赛题

后端仓库请移步[SmartEditor](https://github.com/electronic-pig/SmartEditor)

# 🎉 特性

## 整体功能

- 用户认证
- 文档管理
- 富文本编辑
- AI功能
  
## 赛题要求

- 智能润色
- 多媒体信息提取
- 智能格式排版
